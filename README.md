🎯 소개
Soccer Store는 축구 용품을 쉽고 빠르게 구매할 수 있는 Android 모바일 쇼핑 애플리케이션입니다.
💡 개발 동기

🔍 축구 용품을 찾기 위해 여러 사이트를 돌아다니는 불편함
📱 모바일에 최적화되지 않은 기존 쇼핑 앱들
🎯 카테고리별 검색이 어려운 사용자 경험

✨ 해결 방안
축구 용품만을 위한 전문 모바일 스토어를 통해 직관적이고 빠른 쇼핑 경험을 제공합니다!

🚀 주요 기능
1️⃣ 실시간 검색 🔍

상품명과 설명을 기반으로 즉시 검색
입력하는 즉시 결과 필터링
대소문자 구분 없는 스마트 검색

2️⃣ 카테고리 필터 🏷️

축구화 - 프로급 성능의 축구화
축구공 - 공식 경기용 축구공
유니폼 - 팀 유니폼 및 트레이닝복
장비 - 골키퍼 글러브, 정강이 보호대
훈련용품 - 트레이닝 콘, 마커
가방 - 축구 전용 백팩 및 가방

3️⃣ 장바구니 🛒

원클릭 상품 추가
실시간 수량 조절
자동 총액 계산

4️⃣ 주문 시스템 💰

장바구니 내역 확인
간편한 주문 프로세스
주문 완료 알림
